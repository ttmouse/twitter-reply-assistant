# 首页发布框AI扩写功能使用指南

## 功能简介

Twitter回复助手现已支持在首页发布框中使用AI扩写功能！您可以在发布新推文时，先输入简单的想法或大纲，然后使用AI将其扩展为完整的推文内容。

## 使用方法

1. **访问Twitter首页**
   - 确保您已登录Twitter/X.com
   - 确保您已配置好AI服务商和API密钥

2. **点击"发布"按钮**
   - 点击首页右侧或顶部的"发布"按钮，打开发布框

3. **输入初始内容**
   - 在发布框中输入您的简单想法、大纲或种子内容
   - 例如："今天天气真好，想去公园走走"

4. **使用AI扩写**
   - 在发布框的工具栏中，您会看到一个蓝色的扩写图标（🔽）
   - 点击此按钮，AI将把您的简单想法扩展为更完整、更丰富的推文
   - 等待几秒钟，扩写内容将自动填充到发布框中

5. **编辑和发布**
   - 扩写后的内容完全可编辑
   - 您可以根据需要进行修改和调整
   - 满意后，点击"发布"按钮发布推文

## 功能特点

- ✨ **智能扩写** - 根据您的种子内容生成符合语气的完整推文
- 🎯 **保持原意** - 扩写内容保留您原始想法的核心意思
- 📝 **完全可编辑** - 扩写后的内容可以进行任何修改
- 🚀 **快速高效** - 几秒钟内完成扩写，提高发布效率
- 🎨 **自然流畅** - 生成的内容自然流畅，符合Twitter风格

## 常见问题

**Q: 扩写按钮在哪里？**
A: 扩写按钮位于发布框工具栏中，与媒体、表情符号按钮在同一行，图标是蓝色的扩写箭头（🔽）。

**Q: 为什么没有看到扩写按钮？**
A: 请确保：
- 您已正确配置AI服务
- 您在Twitter首页（不是在回复框中）
- 插件已启用并正常工作

**Q: 扩写内容不符合预期怎么办？**
A: 扩写内容是完全可编辑的，您可以：
- 直接修改扩写后的内容
- 尝试使用不同的种子内容
- 重新点击扩写按钮生成新内容

**Q: 扩写功能支持哪些语言？**
A: 扩写功能支持所有语言，包括中文、英文、日文等。内容将根据您输入的种子语言进行扩写。

**Q: 扩写后的内容长度如何控制？**
A: AI会自动控制扩写内容的长度，确保符合Twitter的字数限制（280字符）。

## 技术说明

- 扩写功能仅在首页发布框中可用，不影响回复框中的AI回复功能
- 扩写功能使用与回复功能相同的AI服务配置
- 扩写内容不会自动发送，需要用户确认后才发布
- 所有数据都存储在本地，不会上传到第三方服务器
