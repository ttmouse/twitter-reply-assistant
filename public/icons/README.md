# Extension Icons

This directory should contain the following icon files:

- icon16.png (16x16px)
- icon32.png (32x32px)
- icon48.png (48x48px)
- icon128.png (128x128px)

## Icon Design Guidelines

- Use a simple, recognizable symbol
- Ensure it works at small sizes (16x16)
- Use Twitter's color scheme for consistency
- Make it distinct from Twitter's native icons

## Temporary Solution

Until proper icons are added, the extension will still load but may show a default placeholder in the browser toolbar.
